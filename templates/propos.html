HangmanWeb - LesNullos

Un projet de première année (B1 Info/Cyber) à Ynov

Présentation du Projet

HangmanWeb est un jeu du pendu sur le web, développé en HTML, CSS, et GoLang.

Prérequis

  • Installation de Visual Studio Code
  • Installation de GoLang
  • Vérification de la version de GoLang : go version (version utilisée : Go 1.23.0)

Présentation du Jeu

Notre version de HangmanWeb est un jeu de pendu sur le thème de Minecraft, avec plusieurs niveaux de difficulté et un tableau des scores.

Organisation du Projet

  • HangmanWeb-Lesnullos : dossier principal
  • main.go : fichier principal contenant le code du jeu
  • mots.txt : liste de mots à deviner
  • templates : fichiers HTML
  • static : dossiers pour les ressources CSS et images

Pour Lancer le Projet

  1. Clonez le projet : git clone https://github.com/jfrbast/HangmanWeb-lesNullos.git
  2. Accédez au dossier : cd HangmanWeb-lesNullos
  3. Lancez le programme : go run .
  4. Ouvrez http://localhost:8080 dans votre navigateur

Amusez-vous !
