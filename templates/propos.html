HangmanWeb - LesNullos

Présentation du Projet

HangmanWeb est un projet de première année (B1 Info/Cyber) à Ynov, dont l'objectif est de créer un jeu du pendu sur le web en utilisant HTML et CSS, avec une programmation en GoLang.

Prérequis

  • Installation de Visual Studio Code : un éditeur de code. Téléchargez-le ici : Visual Studio Code.
  • Installation de GoLang : téléchargez la version de Go via le site officiel : GoLang Installation.
  • Vérification de la version installée : dans un terminal, exécutez la commande suivante :

go version

Pour ce projet, nous avons utilisé la version Go 1.23.0 (windows/amd64).

Présentation du Jeu

Notre version de HangmanWeb est un jeu de pendu basé sur le thème de Minecraft, avec plusieurs niveaux de difficulté et un tableau des scores !

Organisation du Projet

Nous avons structuré notre projet comme suit :

  • HangmanWeb-Lesnullos : dossier principal contenant le projet.
  • main.go : fichier principal contenant le code du jeu du pendu.
  • mots.txt : fichier texte contenant la liste des mots à deviner.
  • templates : dossier contenant les fichiers HTML.
  • static : dossier contenant les sous-dossiers suivants :
    • css : contient les fichiers CSS.
    • images : contient les fichiers image (format .png).

Pour Lancer le Projet

  1. Ouvrez un terminal et clonez le projet avec la commande suivante :

    2. git clone https://github.com/jfrbast/HangmanWeb-lesNullos.git

  2. Accédez au dossier du projet :

    3. cd HangmanWeb-lesNullos

  3. Lancez le programme en exécutant :

    4. go run .

  4. Ouvrez votre navigateur web et entrez l'URL suivante :

    5. http://localhost:8080

  5. Amusez-vous !
